- ## 什么是泛型
	- 泛型，即“泛化类型”。
	- 那“泛化类型”是什么？先从参数说起，我们都知道参数是方法中要传递的值，定义方法填写的是形参，调用方法传入的是实参。泛化类型就是将原来具体的类型抽象化，此时类型也定义成参数形式，然后在使用/调用时传入具体的类型
	- ```
	  //a,b是形参
	  int plusNumber(int a,int b){
	      return a + b;
	  }
	  
	  //调用传入的值为实参
	  plusNumber(1,2)
	  
	  //泛型方法
	  T plusNumber(T a, T b){
	      return a + b;
	  }
	  ```
