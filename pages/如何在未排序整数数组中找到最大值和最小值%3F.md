- 1、用两个变量同时保存最大和最小值，
- 2、然后用一层for循环遍历整个数组，
- 3、在循环过程中对数据进行对比，找出最大和最小值即可。
- ```java
      public int[] getMaxMin(int[] nums){
          // 1、定义2个变量存最大最小
          int min = -1,max = 0;
          //2、遍历数组对比
          for(int num :nums){
              if(num<min){
                  min = num; 
              } 
            	if(num>max){
                  max = num;
              }
          }
          return new int[]{min,max};
      }
  ```
