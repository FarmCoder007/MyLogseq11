- ## 一、简介
	- ASM 是一种通用的Java字节码操作和分析框架，它可以用于修改现有的class文件或者动态生成class文件
	- 处理字节码方式是“拆分-修改-合并”

- ## 三、ASM组成(核心API  树API)
	- 组成结构上来说，ASM分成两部分，一部分为Core API，另一部分为Tree API
	- ### [[核心API]]
	- Tree API
		- 包括asm-tree.jar和asm-analysis.jar
