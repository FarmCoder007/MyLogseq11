- 方法是，快慢指针相遇后继续移动，直到第二次相遇。两次相遇间的移动次数即为环的长度。
- ```java
  public class LinkedListCycleLength {
      public int findCycleLength(ListNode head) {
          ListNode slow = head;
          ListNode fast = head;
          boolean hasCycle = false;
          int steps = 0;
  
          // 检测是否存在环
          while (fast != null && fast.next != null) {
              slow = slow.next;
              fast = fast.next.next;
              if (slow == fast) {
                  hasCycle = true;
                  break;
              }
          }
  
          // 如果存在环，计算环的长度
          if (hasCycle) {
              do {
                  slow = slow.next;
                  fast = fast.next.next;
                  steps++;
              } while (slow != fast);
          }
  
          return steps;
      }
  }
