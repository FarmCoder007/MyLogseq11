# 一、数据结构:
	- Object 数组。内存连续
- # 二、特点：
	- 线程不同步的   （效率高，代替了Vector）
	- 因为数据结构是数组，[[#red]]==**查询效率高，实际通过地址访问**==
- # 三、常用方法：
	- 增：add
	- 删：remove。
	- 改：set。
	- 查：get。
- # 四、判断元素是否相同
	- 对象建在堆中    集合里存的对象都是引用。
	- arraylist
		- 判断元素是否相同arraylist使用的是contains【仅仅使用的是对象的equals方法】，重写对象的equals方法。
	- hashSet
		- 而hashSet使用的是hash表加equals
