- # 2、ActivityStarter.startActivity[2]中返回了START_INTENT_NOT_RESOLVED，START_CLASS_NOT_FOUND
  collapsed:: true
	- ```java
	        //接下来开始做一些校验判断

	            // We couldn't find a class that can handle the given Intent.
