- # 三、JSX语法
  collapsed:: true
	- ## 简介
		- ```js
		  const element = <h1>Hello, world!</h1>;
		  ```
		- 这个有趣的标签语法既不是字符串也不是 HTML。
		- 它被称为 JSX，是一个 JavaScript 的语法扩展。我们建议在 React 中配合使用 JSX，JSX 可以很好地描述 UI 应该呈现出它应有交互的本质形式。JSX 可能会使人联想到模版语言，但它具有 JavaScript 的全部功能。
